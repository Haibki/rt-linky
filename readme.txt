== Description ==

RT-Linky ist ein moderner Link-in-Bio Generator für WordPress. Erstelle schöne Profilseiten mit allen deinen wichtigen Links.

= Features =

* **Custom Post Type** - Profile als native WordPress Posts
* **Moderne React Admin** - React-basierte Oberfläche
* **REST API** - Vollständige REST API
* **Gutenberg Block** - Nativer WordPress Block
* **Echtzeit-Vorschau** - Live-Preview beim Bearbeiten
* **Statistik-Tracking** - Detaillierte Analytics
* **SEO optimiert** - Open Graph und Twitter Cards

= Free vs Pro =

**Free Version:**
* Max. 2 Links
* 2 Icons verfügbar
* Nur Farben/Gradient Hintergründe
* Footer immer aktiviert

**Pro Version:**
* Unbegrenzte Links
* Alle 25+ Icons
* Bild-Upload für Hintergründe
* Link-Untertitel
* Verifiziert-Badge
* Abschaltbarer Footer
* Detaillierte Analytics
* Premium Support

[Jetzt auf Pro upgraden](https://rettoro.de/rt-linky)

== Installation ==

1. Lade das Plugin-Verzeichnis in `/wp-content/plugins/` hoch
2. Aktiviere das Plugin im WordPress-Backend unter "Plugins"
3. Gehe zu "RT-Linky" im Admin-Menü
4. Erstelle dein erstes Profil

== Frequently Asked Questions ==

= Wie viele Links kann ich in der Free-Version erstellen? =

In der Free-Version kannst du maximal 2 Links erstellen. Für unbegrenzte Links upgrade auf [Pro](https://rettoro.de/rt-linky).

= Bleiben meine Links erhalten wenn ich downgrade? =

Ja! Deine bestehenden Links bleiben immer erhalten, auch wenn du von Pro auf Free downgradest oder deine Lizenz abläuft. Du kannst nur keine neuen Links erstellen.

= Gibt es einen Gutenberg Block? =

Ja, RT-Linky enthält einen nativen Gutenberg Block. Du kannst aber auch den Shortcode `[rt-linky id="123"]` verwenden.

= Kann ich mein Profil unter einer eigenen URL aufrufen? =

Ja, Profile sind erreichbar unter `deineseite.de/link/dein-profil-slug/`
